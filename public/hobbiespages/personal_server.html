About Personal Server

Hello! You have stumbled onto my about my server page. The server is made in Linux, I used a GUI version of Ubuntu to create it; and it lays on my home network. My personal server is a few things:Home Entertainment Server, a NAS, a server for some LAN games, and a personal DB that houses my recipes. And on here I can take you on a step by step tutorial on how to get your very own Ubuntu server!

PARTS LIST

  • Computer (new or junk)
    • If using a laptop you won't need a monitor and a keyboard at the very least.
    • You will need a way to display what is on your server's screen until you have it all configured.
    • KEYBOARD and MONITOR are needed.
  • ethernet cable
  • Another computer on your home network to be able to access the server from.
  • Lots of hard disk space. Minimum of 1TB HDD reccomended
    • I used a 1 TB hard disk drive I had laying around for this.
    • I also eventually got a upgraded case and more drives installed.

THE HOW TO

  1. Get a junk computer/laptop that has a lan port
    • It doesn't have to be junk just something that can connect to your modem/wifi spot in your home.
    • I used a computer that I got dirt cheap that still has ddr3 ram in it. But it works great for this.
    • The lan port is preffered so the connection is not interrupted.
  2. Install your linux distro of choice
    • I used a Debian based Linux and Ubuntu.
    • Does not have to be Debian but its what I am used to.
    • I also used the Ubuntu LTS Desktop version and not the server. Either are good, I just like the GUI of the desktop. All it really does is eat some time up and space on hard disk for the server function. Still works fine if you want to use the Server or Desktop iso.
    • DO A CLEAN INSTALL, so it wipes everything on the hard drive
  3. Update all packages in the distro
    • On Debian:
      1. sudo apt-get update
      2. sudo apt-get upgrade
    • On Red Hat:
      1. sudo yum update
  4. Install Samba Service
  5. Configure server to have a static IP Address on your network.
  6. ON A DIFFERENT COMPUTER, install openssh
  7. connect to your server with the Username you added in Samba
